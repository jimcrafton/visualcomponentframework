Building the Microsoft VC++ Wizards and Add-ins

Introduction

This will explain building the Wizards and Add-ins that come with the VCF for Microsoft's Visual C++ 6.x

Win32

Required software:

Microsoft Visual C++ 6.x

The first thing you'll want to is build the various add-ins for Developer Studio. Open the main workspace found in vcf/build/vc60/add-ins/all.dsw. Build the "Win32 Release" configuration, making sure that the "all" project is the current active project. This should build the VCF New Class add-in, plus project wizards VCFLibraryAppWizard, VPLAppWiz, VCFConsoleWiz, and VCFAppWiz.
