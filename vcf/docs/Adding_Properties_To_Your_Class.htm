Adding Properties To Your Class

You can add, edit, or remove properties from your class while the Wizard is running, before you click "OK". If you right click anywhere on the Properties list, you can select from the context menu, as show in the following screen shots, and can either add a new property, or edit or remove an existing property. Each of the menu options is discussed in more detail below.

Click on the "Add..." menu item to add a new property to your class. The new property will include the correct VCF RTTI macro, and the appropriate public member function to get and set the properties value, as well as an appropriate private member variable of the correct type.

Click on the "Edit..." menu item to edit an already selected property in the list. The Property dialog will appear and you can then edit the property as necessary.

To remove a property from the class just select it and click

 

 

The "Name" field displays the name of the property - remember: don't use spaces in the name, otherwise the functions that are generated will not be correct.

The "Type" list shows a list of possible types, such as int, long, char, and String (the VCF's typedef around std::basic_string).

Checking the "Read Only Property" will only generate a get function for your property. By default it is unchecked, meaning you will have get and set functions generated.

Clicking the "OK" button changes or adds your property to the your class.

Clicking the "Cancel" button cancels your changes, or addition of the property.
