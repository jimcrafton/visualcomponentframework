Downloading the VCF

Introduction

You can download the VCF from SourceForge. There are a variety of files, such as documentation, source, installers, all of which can be seen by following this link.

For source code you can download the latest source code in a compressed archive, usually of the form vcf<version>.tar.gz (for example the 0.5.4 version is named vcf.0.5.4.tar.gz). 

Alternatively you can download the code anonymously through CVS. You'll have to install CVS on your system first and then once this is done follow these instructions:

The module you wish to check out must be specified as vcf for the whole thing. When prompted for a password for anonymous, simply press the Enter key.

cvs -d:pserver:anonymous@cvs.vcf.sourceforge.net:/cvsroot/vcf login

cvs -z3 -d:pserver:anonymous@cvs.vcf.sourceforge.net:/cvsroot/vcf co vcf

This will pull down the whole framework and related 3rd party modules. 

Lastly, you get the framework installed by downloading the VCF Installer, named VCFInstaller<version>.exe, which will set up your system with the VCF and allow you to immediately be ready to start developing with the VCF. This includes setting up any environment variables, MSDN integration of documents (if requested), and registering of the Visual C++ add-ins and VCF application wizards (for Win32 installs only).
