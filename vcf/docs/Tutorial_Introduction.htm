Tutorial Introduction

This is a series of tutorials designed to bring you up to speed in how to use the VCF.

The first tutorial will cover the basics of creating a class that supports the basic features of the VCF, and how you can easily save the state of your class using the VCF stream classes.

The second tutorial will build on the first, and demonstrate how to use the advanced RTTI features of the VCF and what this can do for you.

The third tutorial will demonstrate the basics of events in the VCF, how they work, and how to create events of your own.

The fourth tutorial will deal with creating a simple UI, and how the various controls and components work together.
