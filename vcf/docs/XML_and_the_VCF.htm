XML and the VCF

The VCF uses XML as it storage format for user interface windows. Any top level window can persist itself using ComponentOutputStream's and a ComponentInputStream's. To implement the XML parsing the VCF currently uses the Xerces XML library that is freely available from the xml.apache website.

As of this writing the current version of Xerces being used is 1.3, if you suspect you have an older version please goto either the VCF site for a zip file of the correct version of the binaries, or see the Xerces site itself.
