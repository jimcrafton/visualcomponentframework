Required Tools

Introduction

There are several tools you'll need before you can start developing with the VCF. 

Compilers

Here is a list of compilers for various platforms

Platform Compilers
Win32 VC++ 6, GCC 2.95 or better
Linux GCC 2.95 or better
Solaris (2.8, although this may hold true for earlier versions)  GCC 2.95 or better

 

IDE's 

Currently the IDE that I have any experience working with VCF under is VC6

[other users can mention other uses of the VCF in other IDES]

SourceControl

The VCF uses CVS (Concurrent Versions Control ) for version control. On top of this, for security, SSH is used to encrypt the data sent via cvs.
