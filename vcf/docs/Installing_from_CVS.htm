Installing from CVS

Setup

To begin installing the VCF from source control you'll need access to CVS, a popular open source version control system. You'll need to install this on your system in order to connect to the CVS server that is hosted by SourceForge, so you can pull down the latest source code. You can find out more about installing CVS and setting it up here.

CVS and SSH

When using CVS with a SourceForge project, SSH is used to ensure a secure connection. Because of this you'll need SSH on your system too. Because both are needed, when installing the VCF, the best way is to use the VCF Installer which will automatically configure your system for using CVS and SSH.
