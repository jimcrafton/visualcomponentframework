Tutorial 4

This tutorial will focus on how to write a simple application using the VCF.

Creating windows and controls in the VCF is extremely simple so let's dive write in and look at a sample "Hello World" program that displays a window with "Hello World" in it's title bar.

1  int main( int argc, char *argv[] )
2  {
3      Application app;
4      Window* mainWindow = new Window();
5
6      mainWindow->setCaption( "Hello World" );
7
8      app.setMainWindow( mainWindow );
9
10     Application::appMain( argc, argv );
12
13     return 0;
14  }

Cool ! Our first Window in only a few lines of code! The first thing we do is to create an Application instance on the stack call app (line 3). We then create a window by creating a new instance of a Window class and call it mainWindow(line 4). This is always done on the heap, using the new operator. There are a variety of reasons for this, but for now just take my word for it (we'll look into it later). Next we set the Window's caption via the setCaption() method, and pass in the string "Hello World" (line 6). The Application's setMainWindow() is called and the mainWindow is passed in (line 8). Doing this ensures that the Application has a pointer to a main window, which in turn allows the Application to be notified when the main window closes, which in turn shuts down the Application. The final step is to invoke the static method Application::appMain(), passing in the arguments that the main() function received. This method starts the Application, which will continue to run until it's main window is closed.

Now that wasn't too painful was it ?
